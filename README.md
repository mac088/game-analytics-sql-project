# game-analytics-sql-project
# Аналитический проект по игре "Секреты Темнолесья"

Проект представляет собой комплексный SQL-анализ игровых данных RPG "Секреты Темнолесья". Включает исследовательский анализ данных и решение ad-hoc задач от маркетинговой команды.

## Структура проекта

### Часть 1. Исследовательский анализ данных

**Задача 1. Исследование доли платящих игроков**
- Расчет общей доли игроков, покупавших внутриигровую валюту "райские лепестки"
- Анализ зависимости доли платящих игроков от выбранной расы персонажа

**Задача 2. Исследование внутриигровых покупок**
- Статистический анализ стоимости покупок эпических предметов
- Выявление аномальных покупок с нулевой стоимостью
- Анализ популярности эпических предметов среди игроков

### Часть 2. Решение ad hoc задач

**Задача: Зависимость активности игроков от расы персонажа**
- Сравнение игровой активности между разными расами персонажей
- Анализ метрик: количество покупок, средняя стоимость, доля платящих игроков
- Проверка гипотезы о балансе сложности игры для разных рас

### Часть 3. Выводы и аналитические комментарии
- Ключевые инсайты по каждой задаче
- Рекомендации для маркетинговой команды
- Выводы о взаимосвязях между характеристиками персонажей и purchasing behavior

## Техническая реализация

- **База данных:** PostgreSQL
- **Инструмент:** DBeaver
- **Методология:** SQL-анализ с комментариями к каждому запросу
- **Документация:** Google Документы (шаблон проекта)

## Ключевые метрики анализа

- Доля платящих игроков (Paying User Rate)
- ARPPU (Average Revenue Per Paying User)
- Статистические показатели стоимости покупок
- Популярность игровых предметов
- Активность игроков по расам персонажей

## Цель проекта

Предоставить data-driven insights для:
- Балансировки игрового процесса между расами
- Оптимизации монетизации
- Улучшения маркетиных стратегий
- Повышения вовлеченности игроков
